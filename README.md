## 🚀 如何运行？

1. 下载整个文件夹
2. 双击 `index.html`
3. 浏览器会打开一个页面，如果长得像京东，那是 AI 厉害；如果不像，那是 AI 发挥失常。
4. 如果页面没加载出来，请检查是不是图片路径错了 —— 反正 AI 写的，我不负责。
